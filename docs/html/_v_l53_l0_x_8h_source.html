225// Sets the return signal rate limit to the given value in units of MCPS (mega counts per second).
226// This is the minimum amplitude of the signal reflected from the target and received by the sensor
227// necessary for it to report a valid reading. Setting a lower limit increases the potential range
228// of the sensor but also increases the likelihood of getting an inaccurate reading because of
